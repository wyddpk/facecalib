一人一档程序
版本：1.0.0
环境：tensorflow==1.7.0（GPU）tensorlayer==1.6.0 sklearn==0.19.2
功能：将文件夹中所有头像进分类
方式：首先在指定文件夹中新建一个文件夹，放入一张图片，将该图片当做个人特征字典的第一个key，之后对要处理的文件夹中每一个头像进行对比，如果电脑认为是同一个人，
      则将该头像放入认为同一个的文件夹，并更新这个人的字典，如果电脑认为不是同一个人，则新建一个文件夹，将该图片放入，在字典中新增这个人的key和特征矩阵。
      （当两人相似距离为0.6以上则认为是同一个人，小于0.6当做其他人）
